**LegelX** is a state-of-the-art mobile intelligence platform designed to bridge the gap between complex legal jargon and human understanding. Powered by the world's most advanced AI models (Gemini 3), LegelX provides real-time contract analysis, risk assessment, and a simulated AI Courtroom for strategic preparation.

---

## 💎 Full Feature Catalog

### 🧠 1. AI Intelligence Suite (Gemini 3 Powered)
The brain of LegelX uses a multi-layered reasoning engine to process legal documents with human-level accuracy.
*   **Precision Summarization**: Converts 50-page documents into 5-point actionable summaries.
*   **Risk Detection Engine**: Automatically scans for termination clauses, indemnification traps, and non-compete overlaps.
*   **Fairness Indexing**: A proprietary algorithm that scores a contract from 0-100 based on bilateral benefit.
*   **Multi-Model Fallback**: Seamlessly switches between Gemini 3 Pro and Flash to ensure 99.9% uptime.

### 🏛️ 2. The AI Courtroom Experience
A unique strategic tool for lawyers and law students to rehearse cases.
*   **AI Judge Persona**: Provides unbiased rulings based on provided evidence and contract clauses.
*   **AI Prosecutor/Opposing Counsel**: Dynamically finds contradictions in your arguments to help you prepare better.
*   **Interactive Testimony**: Chat-based simulation of cross-examinations.
*   **Strategic Verdicts**: Get a detailed breakdown of why you won or lost the simulated session.

### 🔍 3. Advanced Document Processing (OCR+)
Legacy paper contracts are no longer a barrier.
*   **Neural OCR**: Scan physical documents with your camera; LegelX extracts text with 98% accuracy.
*   **PDF Intelligence**: Native support for complex PDF structures, tables, and fine-print footers.
*   **Multi-Document Context**: Chat across multiple uploaded documents to find conflicting clauses between different agreements.

### 📊 4. Analytics & Dashboard
Visualizing data is key to quick decision-making.
*   **Intelligence Breakdown**: Interactive charts showing clarity, compliance, and enforceability metrics.
*   **Global Trend Tracking**: See how your current contract compares to industry standard benchmarks.
*   **Recent Activity Feed**: Instantly jump back into your last analyzed cases or courtroom sessions.

### 🎨 5. Premium UI/UX Design
Built for high-performance professionals.
*   **Neumorphic & Glassmorphic UI**: A stunning, modern interface with depth and clarity.
*   **Real-time Animations**: Smooth transitions and micro-interactions for a fluid experience.
*   **Optimized Performance**: Zero-lag scrolling and instant AI response streaming.
*   **Dark Mode Ready**: Fully responsive theme for late-night legal research.

### 🔐 6. Secure Infrastructure
Data privacy is our top priority.
*   **Supabase Security**: Industry-standard encryption for all user data and documents.
*   **Private AI Processing**: Your data is processed through secure API tunnels and never stored on public training sets.
*   **Auth Gates**: Secure login and multi-factor authentication (MFA) support.

## 🚀 One-Minute Quick Start

Whether you are a coder or a legal enthusiast, getting LegelX running is simple.

### 1. Prerequisites
- **Flutter SDK**: [Install Flutter](https://docs.flutter.dev/get-started/install)
- **Dart**: Included with Flutter.

### 2. Physical Setup
Clone the repository and enter the directory:
```bash
git clone https://github.com/your-username/LegelX.git
cd LegelX
```

### 3. Configuration
Create a `.env` file in the root directory and add your credentials:
```env
# AI & Backend Credentials
GEMINI_API_KEY=your_google_ai_studio_key
SUPABASE_URL=your_project_url
SUPABASE_ANON_KEY=your_anon_key
```

### 4. Deploy
Run the application on your physical device or emulator:
```bash
flutter pub get
flutter run
```
